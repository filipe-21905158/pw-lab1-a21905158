    HTML5 e CSS:

      O que e HTML5

      E uma linguagem usada para estruturar r apresentar contiudos. E a quinta e mais recente versao do HTML(suportada pela apple, google, mozila e microsoft)

      HTML
      Elemento Utilizado Função
      Figure ✔️ permite agrupar um elemento unico com uma legenda
      Iframe ✔️ permite incorporar outro documento no documento HTML atual
      Head ✔️ define o cabecalho do documento
      Body ✔️ Define o corpo do documento
      footer ✔️ define o rodapé do documento
      Hx ✔️ define cabeçalhos/titulos
      p ✔️ define um paragrafo
      br ✔️ define uma linha em branco/enter
      hr ✔️ define uma linha continua ao longo da pagina
      table ✔️ define uma tabela
      tdead ✔️ utilizado para agrupar um header de certa coluna
      tbody ✔️ contem a parte principal da tabela(contem a maioria da info)
























      O que e CSS

      E uma linguagem de estilo utilizada para descrever a apresentacao de um documento escrito em uma linguagem como o html

      Tipos de seletores
      Seletor Utilizado Função
      Universal ❌ aplica o estilo a todos os elementos da pagina
      Tipo ❌ usa o nome do marcador estilizado
      Classe ❌ associa um nome a um estilo
      Id ❌ permite identificar univocamente um elemento
      Atributo ❌ aplicam-se atributos a elementos
      Pseudo-classe ❌ associado ao estado de um elemento
      Pseudo-elemento ❌ aplicam-se a partes de elementos

















      Combinsdores de Seletores
      Combinadores Utilizado Função
      Descendentes ❌ são sepadados por espaço
      Filhos ❌ Através do uso de ">"
      adjacentes ❌ Através do uso de "~"
      imediatamente adjacentes ❌ Através do uso de "+"
